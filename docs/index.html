Installation & Updating

A computer or device with the following is required: 32-64-bit CPU 32-64-bit operating system 1GB RAM or better

Installing/updating manually

* Windows:

Make sure to have Python installed as it is required to run PieMC! Make a folder for your server. Download the latest stable version of PieMC on github. Double click on start.cmd


* Linux:

Make sure to have Python installed as it is required to run PieMC! Make a folder for your server. Download the latest stable version of PieMC on github. Run ./start.sh

Plugins

PieMC can be extended and customized with plugins. A plugin is an external module which can be ‚Äúplugged in‚Äù to your server to add custom features, change default behaviours and more.

Downloading Plugins

You can find a wide range of premade plugins on PieMC-Plugins our release plugin platform.

Installing/updating a plugin

Plugin developers usually distribute plugins as .zip files. These files can be easily loaded by placing them in your plugins folder in the server data folder. Be sure to delete old versions of the plugin, and restart the server to see changes.
